Davi's Minecraft Server

What is this?

I'm experimenting with an always-up minecraft server for family and friends. If you know me, then you're good to join!

The server will be running on my old macbook pro, so it won't be incredibly powerful, but it'll be enough for us to have some fun and for me to test my technological abilities.

Etiquette

Be excellent to each other. Don't punch people, take stuff from their homes, break their builds, etc unless you have their explicit consent. I expect we'll have some combination of working together on one megabase and us each having our own base.

Use nice language. This means that you should do your best to abide by "no swearing in my good Christian Minecraft server." It also means that you should use correct pronouns, refrain from insults and prejudice, and use language to appreciate and include others.

You can play by yourself whenever you want and/or organize times to be playing with others; feel free to play when I'm not. If you notice a problem, whether it be griefing or that the server is down, please let me know.

Settings

  • Minecraft Java version 1.20.1
  • survival
  • difficulty hard
  • mob griefing turned off

Mods

Botania

This is the biggest mod here. It adds a tech tree, automation tools, and magic. It has an in-game guide craft-able by combining a book and a sapling.

Quark (including oddities)

This adds a lot of subtle features to make minecraft better. You may wish to skim the full feature list on the website. Some highlights are

  • Pipes
  • Tetris-like enchanting mechanics
  • Drop a little icon that holds all your stuff for you when you die rather than your stuff going BLEARGH all over.
  • Inventory sorting and hot-deposit into chests
  • Rope (deployable ladders that also move blocks)
  • Right click a crop to harvest+replant. Use a hoe to harvest+replant large area.
  • Ancient tomes (rare loot that lets you enchant things even more)

Waystones for teleporting

EMI for help crafting things

Appleskin for seeing hunger saturation

Explorer's Compass and Nature's Compass for finding biomes and structures

Curios for equipping accessories (g)

Trade Cycling for getting good villager trades

Disenchanting for moving enchantments from items to books

Light overlay for seeing where to place torches (F7)

There are other mods too, but these are the main ones that you benefit from knowing about.

Installation

Prism

This is the recommended method. Prism is an open source alternative Minecraft launcher

  1. Download Davi's Server 2024.zip from my google drive
  2. Download and install Prism
  3. Open Prism
  4. Open the account manager and follow the process to add your Microsoft account (the one you use to log in to Minecraft)
  5. Add Instance -> Import -> browse to Davi's Server 2024.zip
  6. If you need Java, download a JDK such as Amazon Corretto
Alternate installation instructions (Manual)
  1. Start up the minecraft launcher, select version 1.20.1 (to the left of the play button), and run the game to make sure it is downloaded
  2. Download the latest stable version of Forge for Minecraft 1.20.1 and follow its instructions to install client
  3. Start up the minecraft launcher, select version 1.20.1-forge (to the left of the play button), and run the game to let Forge generate its necessary files and folders. You may desire to leave the game running for the next step.
  4. Locate your mods directory within the directory from which Minecraft runs. An easy way to do this it to (in the game's main menu) click on Mods and then Open mods folder
  5. Download all the jar mod files (and extract out of the zip file if applicable) from my Google Drive
  6. Move the mods into your mods folder
  7. Close Minecraft if it's still open. Within the minecraft launcher, specify the correct minecraft/forge version (to the left of the play button)
