Davi's Minecraft Server

What is this?

I'm experimenting with an always-up minecraft server for family and friends. If you know me, then you're good to join!

The server will be running on my old macbook pro, so it won't be incredibly powerful, but it'll be enough for us to have some fun and for me to test my technological abilities.

Etiquette

Be excellent to each other. Don't punch people, take stuff from their homes, break their builds, etc unless you have their explicit consent. I expect we'll have some combination of working together on one megabase and us each having our own base.

Use nice language. This means that you should do your best to abide by "no swearing in my good Christian Minecraft server." It also means that you should use correct pronouns, refrain from insults and prejudice, and use language to appreciate and include others.

You can play by yourself whenever you want and/or organize times to be playing with others; feel free to play when I'm not. If you notice a problem, whether it be griefing or that the server is down, please let me know.

Settings

This will be a survival server, difficulty hard, with mob griefing turned off.

How do I join?

Minecraft version 1.20.1

You will need Minecraft Java version 1.20.1 (sorry bedrock folks). You can install this from the Minecraft launcher.

Mods

The server will use the mods listed below, most of which you will need to install before you can join. You may choose to download each one individually, but you'll probably be better off if you download all but Forge from my google drive. So in general, the instructions are

  1. Start up the minecraft launcher, select version 1.20.1 (to the left of the play button), and run the game to make sure it is downloaded
  2. Download the latest stable version of Forge for Minecraft 1.20.1 and follow its instructions to install client (I used Neoforge 1.20.1-47.1.88)
  3. Start up the minecraft launcher, select version 1.20.1-forge (to the left of the play button), and run the game to let Forge generate its necessary files and folders. You may desire to leave the game running for the next step.
  4. Locate your mods directory within the directory from which Minecraft runs. An easy way to do this it to (in the game's main menu) click on Mods and then Open mods folder
  5. Download all the mods (and extract out of the zip file if applicable)
  6. Move the mods into your mods folder
  7. Close Minecraft if it's still open. Within the minecraft launcher, specify the correct minecraft/forge version (to the left of the play button)

Forge

Neoforge also seems to work.

Botania

This is the biggest mod here. It adds a tech tree, automation tools, and magic. It has an in-game guide craft-able by combining a book and a sapling. You can read more about it on the website.

Quark

This adds a lot of subtle features to make minecraft better. You may wish to skim the full feature list. Some highlights are

  • Inventory sorting and hot-deposit into chests
  • Rope (deployable ladders that also move blocks)
  • More mobs
  • Torch arrows
  • Ancient tomes (rare loot that lets you enchant things even more)
  • Right click a crop to harvest+replant. Use a hoe to harvest+replant large area.
  • Villagers follow emeralds like cows follow wheat
  • Monster boxes are scattered across the underground to make things spicy

The 1.20.1 version is available for download from their discord server (or my google drive)

Quark Oddities

These are a few extra quark features that require a separate file to run because they're more different from vanilla, unmodded minecraft. It includes copper pipes, magnets, and dropping a little icon that holds all your stuff for you when you die rather than your stuff going BLEARGH all over.

Waystones

Waystones are teleport points. Very useful! When you get curious, look at the wiki.

JEI (Just Enough Items)

This lets you see how to craft things! Basic controls (in your inventory) are

  • Show Recipes: Hover over item + R
  • Show Uses: Hover over item + U
  • Toggle Item List Overlay: Control + O

Appleskin

This changes the hunger display to better show the underlying game mechanics.

Explorer's Compass and Nature's Compass

These let you craft special compasses to help you find things.

Curios

You have extra equipment slots accessed by hitting "g". The equipment for those slots comes from Botania.

Trade Cycling

Adds a button to villager GUI that does the equal process of them losing and regaining their profession.

Disenchanting

Adds a block that moves enchantments form items to books.

Dependencies

These code library mods are required by other mods we're using, but don't do anything in and of themselves.

  • Ferrite Core (actually, this just boosts performance, isn't a dependency of other mods)
